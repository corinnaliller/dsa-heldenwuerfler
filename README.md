# dsa-heldenwuerfler
A program to view and roll the dice for a character on the German pen-and-paper game The Dark Eye.

Dieses Programm dient allein dem privaten Gebrauch von DSA-Spielern zur Darstellung von bereits existierenden Helden-Charakteren und zur Durchführung von Talent- und Eigenschaftsproben für diesen Helden.
